# Snake Game 🐍

간단한 HTML/JavaScript로 만든 Snake(스네이크) 게임입니다.

이 프로젝트는 Helpy Chat과 Github MCP(Managed Code Platform)를 활용하여 자동으로 생성되었습니다.

## 주요 기능
- 벽에 부딪히면 게임 오버
- 점수 랭킹 TOP 5 기록
- 속도(Easy/Normal/Hard) 조절
- 시작/재시작 버튼

## 실행 방법
1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 파일을 웹 브라우저로 열면 바로 게임을 즐길 수 있습니다.

## Github Pages로 배포하는 방법
Github Pages를 사용하면 별도의 서버 없이도 웹에서 바로 게임을 플레이할 수 있습니다.

### 1. 파일 구조 확인
index.html과 snake.js가 저장소의 최상위(혹은 원하는 폴더)에 위치해야 합니다.

### 2. Github Pages 설정
1. 저장소의 Settings(설정) 메뉴로 이동합니다.
2. "Pages" 또는 "GitHub Pages" 섹션을 찾습니다.
3. "Source"에서 배포할 브랜치(main)와 폴더(root 또는 /snake-game)를 선택합니다.
   - 이 프로젝트는 `/snake-game` 폴더에 있으므로 `/snake-game`을 선택하세요.
4. 저장하면, 배포 주소가 생성됩니다. (예: `https://사용자명.github.io/snake-game/snake-game/`)

### 3. 배포 주소로 접속
설정 후 안내된 주소로 접속하면 바로 웹에서 Snake 게임을 할 수 있습니다!
